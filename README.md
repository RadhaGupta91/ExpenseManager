<!-- To run application locally -->

git clone (https://github.com/RadhaGupta91/ExpenseManager.git)
cd ExpenseManager
`npm install`
`npm start`

App will automatically get open in browers, If not, then
Open http://localhost:3000 to view it in your browser.

To run Tests
`npm test`
